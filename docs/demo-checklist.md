# Demo Checklist

Короткий чек-лист ручной демонстрации проекта (3-5 минут), если преподаватель попросит показать функционал по шагам.

## 1. Главная и навигация

1. Открыть `/`.
2. Проверить наличие верхнего меню: `Главная / Статьи / Тест / Результаты`.
3. Перейти в `/articles`.

## 2. Статьи и inline-режим

1. Открыть `/articles/js-core`.
2. Прокрутить материал и показать inline-упражнение в тексте.
3. Ввести ответ и нажать `Проверить`.
4. Показать feedback + explanation.

## 3. Test-режим

1. Перейти в `/test`.
2. Ответить на 1-2 вопроса.
3. На одном вопросе нажать `Не знаю, пропустить вопрос`.
4. Дойти до статуса `Тест завершен`.

## 4. Результаты

1. Открыть `/results`.
2. Показать общий score и процент.
3. Показать `Пропущено` у пропущенного вопроса.
4. Показать блок `weak topics`.
5. Нажать `Пройти заново`.

## 5. Тех-проверка перед показом

1. Выполнить:
   - `npm run lint`
   - `npm run test`
   - `npm run build`
2. (Опционально) Выполнить e2e:
   - `npx playwright install chromium`
   - `npm run test:e2e`
